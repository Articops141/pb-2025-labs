1)Создайте публичный репозиторий на GitHub с названием `pb-2025-labs`
ㅤ

2)Клонируйте созданный репозиторий на компьютер
ㅤ

3)Создайте структуру репозитория:

  ** `README.md` - описание вашего репозитория

  ** `labs` - папка с лабораторными работами

  ** `labs/lab1` - папка с файлами вашей первой лабораторной работы

  ** `labs/lab2` - папка для вашей второй лабораторной работы

  ** `...` - аналогичные папки для остальных лабораторных работ


4)Для первой лабораторной работы создайте файл `README.md` с описанием (скопировать условия задания)
ㅤ

5)Сделайте коммит
ㅤ

6)Проделайте то же самое для остальных лабораторных работ. Для каждого файла делайте отдельный коммит
ㅤ

7)Создайте новую ветку `add-picture` и переключитесь на нее
ㅤ

8)В папку с шестой лабораторной работой добавьте любую картинку (цензурную) и сделайте коммит
ㅤ

9)Вернитесь на главную ветку `main`
ㅤ

10)В папку с шестой лабораторной работой добавьте файл `picture.md` и напишите в нем краткое описание вашей картинки
ㅤ

11)Сделайте `merge commit` из `add-picture` в `main`
ㅤ

12)Отправьте изменения в репозиторий, если не делали этого ранее
ㅤ
